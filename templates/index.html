Score Calculation

How The Main Leaderboard Score Is Computed

Each contest score is normalized as \(\text{solve}_{p,i} / \max\_\text{solve}_i\). Then we take the top \({{ k_percent }}\%\) from Saturday and Monday separately.

\[ ext{Score}_p = \left( \frac{w_s\,\sum\left(\text{top}_s\right) + w_m\,\sum\left(\text{top}_m\right)}{w_s\,n_s + w_m\,n_m} \right) \times 100 \]

Where \(n_s = \lfloor {{ k_percent }}/100 \times {{ saturday_total }} \rfloor = {{ saturday_take }}\) and \(n_m = \lfloor {{ k_percent }}/100 \times {{ monday_total }} \rfloor = {{ monday_take }}\). We use \(w_s = {{ saturday_weight }}\) and \(w_m = {{ monday_weight }}\).
